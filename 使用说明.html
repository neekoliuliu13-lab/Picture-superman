组图喵 · 使用说明

把多张评审截图按模版自动组图，支持文字编辑、拖动换位、箭头标注，一键导出高清 JPG / PNG。

★这是什么

一个单文件网页工具（组图喵.html）。所有代码已内联，无需联网、无需安装、不上传服务器，图片只在你本机浏览器里处理。

  • 跨平台：Mac / Windows / iPad / 安卓，任意现代浏览器（Chrome / Edge / Safari）双击即可打开。
  • 它是「工具」不是「存档」：刷新或换设备不保留上次内容，成果请用「导出」存成图片留底。

1快速上手

  1. 上传图片：点左侧「① 上传图片」区域选择文件，或把图片拖进去、Cmd/Ctrl+V 粘贴，支持一次多张。
  2. 填文字：在「② 文字内容」里写标题、底部备注；每张图的标注在下方列表逐条填（可留空）。
  3. 调版式：在「③ 排版参数」里改单行图数、图片宽度、间距、颜色、标题字体等。
  4. （可选）画箭头：在「④ 箭头连线」里开启，标注交互链路。
  5. 导出：右上角选 JPG/PNG 与清晰度，点「一键导出高清」下载图片。

2在预览区直接编辑

✏️ 点字即改在右侧预览里直接点标题 / 图片标注 / 底部备注，原地弹出输入框修改。
回车 保存标题，多行用 ⌘/Ctrl+回车，Esc 取消。
↔ 拖图换位在预览里按住图片拖动，出现蓝色插入条，松手即重排；左侧列表顺序同步（列表里的拖拽手柄也能排序）。

3箭头连线（可选）

用于说明交互链路，默认不画，按需使用。在「④ 箭头连线」面板：

  • 点 ✏️ 开始画箭头 进入画箭头模式，在预览区按住拖动即可画一条。
  • 箭头样式：直线箭头（两点直连）或 弯折箭头（自动走直角折线）。
  • 选中箭头后：拖两端白点调起止点；弯折箭头拖中间白点调折点位置。
  • 可设箭头颜色 / 粗细；按 Delete 删除选中箭头，清除全部箭头 一键清空，Esc 退出模式。
建议先排好图、定好版式，最后再画箭头，避免改版式后箭头位置偏移。

4导出

  • 格式：JPG（体积小）或 PNG（无损、可透明边缘）。
  • 清晰度：标准 2x / 高清 3x / 超清 4x，数值越大越清晰、文件越大。
  • 导出的图片包含所有内容（含箭头），但不含编辑时的选中把手。

5传到另一台设备

把工具文件发到目标设备即可：微信 / 飞书 / 邮件，或拷到 U 盘 / 云盘（iCloud、坚果云等）。

组图喵.html

在新设备上双击该文件 → 浏览器打开 → 直接使用，无需任何额外环境。

iPad / 手机触屏也基本可用（拖图、画箭头）。若触屏体验不顺，可以反馈，我再加触摸事件适配。
